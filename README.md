人間の「思考」と「理解」をテーマに、

**ノーコード教材エディタによる学習データ分析** の2軸で研究しています。

---

## 🧠 主要プロジェクト

### 🔷 **Thinking Fluid Model — 思考の流体モデル（Julia × GLMakie）**
思考を“流体場”として扱うモデルを構築し、  
6層構造（E / R / S / L / V / M）を流体的相互作用として可視化しています。

- Julia + GLMakie による3Dビジュアライズ  
- 感情・価値・傾向の関数化（Value–Tendency Quantification Model）  
- ナビエ–ストークス方程式を参考にした非線形動態モデル  
- 現象学・アーレント思想を土台にした“観察としての思考”の理論化  

👉 **[`thinking_fluid_model/`](./thinking_fluid_model/)**

---

### 🔷 **No-code Education Editor — ノーコード教材エディタ（HTML/JS/Node）**
教育現場向けに、ブラウザだけで動作する  
**教材作成・Glossary・学習ログ分析ダッシュボード** を統合したWebアプリを開発。

- クイズ／診断コンテンツをノーコードで生成  
- 用語辞書（Glossary）と概念ベクトル分析  
- 反応時間・クラスタリング分析などの学習データ解析  
- 教育工学会で発表予定  
- フルスタック構成（HTML / CSS / JS / Node.js）

👉 **[`nocode-site/`](./nocode-site/)**
